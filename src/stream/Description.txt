Ленивая потоковая обработка данных


int res = Stream(1, 2, 3, 4, 5) 			// создаём последовательность от 1 до 5
    | map([](int a) { return a * a;  }) // применяем лямбду-функцию возведения в квадрат для каждого элемента
    | sum(0);							// суммируем все элементы (инициализируем сумму нулём на всякий случай, вдруг поток данных пуст)


Конечно, мы можем записать это и таким образом:

for (int i = 1; i <= 5; i++) {
    res2 += i * i;
}

И второй кусок кода выполнится быстрее. В чём же тогда преимущество?
